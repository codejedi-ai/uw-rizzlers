# UW Rizz Lords 💫

Welcome to the **UWaterloo Rizz Lords** website — a reincarnation of Rizz Club and UW Social.
Built with **Preact + Vite**, this site is more than code: it’s a community bulletin board for charisma, connection, and creativity at the University of Waterloo.

---

## ✨ About Us

We are **not pickup artists**.

* Pickup artists treat love as a game.
* We believe love is **sacred**, and human connection is **divine**.

For us, **RIZZ** means:

* **R**elationships → building bonds that matter
* **I**nitiative → empowering members to lead events
* **Z**est → bringing energy, joy, and creativity
* **Z**en → grounding connections in respect and authenticity

We use rizz not to conquer, but to **connect**.

---

## 🎯 Mission

* Empower members to **host their own events** without relying on a centralized executive team.
* Create spaces where students can **meet, laugh, and grow** together.
* Celebrate the art of being social — authentically, playfully, and respectfully.

---

## 🛠 Features of the Site

* **Community Bulletin Board**: A digital corkboard (powered by our direct-manipulation Preact component) where members can post, drag, and organize events.
* **Events Feed**: A clear list of upcoming socials, study jams, and creative gatherings.
* **Community Hub**: Pages for About, Members, and Gallery.

---

## 💻 Development

This site is built with [Preact](https://preactjs.com/) and [Vite](https://vitejs.dev/).

### Scripts

* `npm run dev` → Start a dev server at [http://localhost:5173/](http://localhost:5173/)
* `npm run build` → Build for production into `dist/`
* `npm run preview` → Preview production build at [http://localhost:4173/](http://localhost:4173/)
